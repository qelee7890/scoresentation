사용법

음표 입력·수정

  • 호버로 회색 음표 미리보기 후 클릭하면 생성
  • 기존 음표 클릭: 박자 증가 · Ctrl+클릭: 박자 감소
  • 더블클릭으로 음표 삭제
  • 음표 위에서 높낮이를 바꿔 호버 후 클릭: 음정 변경
  • 기존 음표는 드래그해서 음높이 변경
  • 기존 음표 클릭: 인라인 메뉴 열기
    • </>: 이 음표부터 줄 끝까지 한 칸 당기기/밀기 (앞 칸이 비어 있어야 당길 수 있음)
    • ♩: 박자 변경 · •: 점음표 토글
    • ♯/♭/♮: 임시표 토글 (조표와 중복되는 항목은 비활성화)
    • 🗑: 음표 삭제

정합성 표시

  • 가사 글자에 매칭되는 음표가 없으면 글자가 빨간색으로 강조됨
  • 가사 길이를 넘어선 음표(짝 없는 dangling 음표)는 줄 끝 오른쪽에 빨간색으로 표시됨
  • 가사 텍스트를 수정해도 dangling 음표는 보존됨 — < 버튼으로 다시 가사 위치로 당기거나 🗑로 정리

조표

  • 음자리표 오른쪽 조표 영역: 호버 후 클릭으로 조표 추가
  • 조표 영역 Ctrl 호버: 샵 미리보기
  • 조표 영역 더블클릭: 마지막 조표 삭제

연결선 (빔)

  • Ctrl/Shift+클릭: 같은 줄에 음표 추가 선택
  • 빈 오선지 드래그: 같은 줄의 음표를 박스로 선택
  • 선택 후 나타나는 인라인 메뉴에서 묶기/해제/취소
  • 연결선 선택은 한 줄씩만 유지됨

가사 편집

  • 가사 줄을 직접 클릭해서 수정
  • 가사 줄에서 Enter: 현재 줄을 둘로 나누기
  • 가사 줄에서 Ctrl+Enter: 현재 위치 이후를 새 슬라이드로 분리
  • 줄 시작에서 Backspace, 줄 끝에서 Delete: 인접 줄 합치기

슬라이드 조작

  • 캔버스에서 우클릭: 슬라이드 추가 · 앞 슬라이드와 통합 · 슬라이드 삭제 메뉴
  • 툴바 아이콘: 슬라이드 복사 / 붙여넣기
  • 악보 블록 좌우 화살표 버튼으로 이전/다음 슬라이드 이동
  • 삭제: 내용을 버리고 슬라이드만 제거 · 통합: 현재 슬라이드 내용을 앞 슬라이드 뒤에 이어붙임

절·후렴 변경

  • 슬라이드 상단 [N절] 뱃지 클릭: 다른 절로 이동 또는 후렴으로 이동
  • 슬라이드 상단 [N번] 뱃지 클릭: 같은 섹션 내 다른 번호 슬라이드와 교체

검색·저장

  • 좌측 곡 검색에서 제목 또는 가사로 최대 10개까지 검색
  • 우측 상단 아이콘: 곡 저장 / 저장본 삭제 / 실행 취소 / 다시 실행
  • Ctrl+Z: 실행 취소, Ctrl+Y: 다시 실행
  • python server.py로 열면 저장/불러오기가 SQLite DB를 우선 사용
