사용법
  • E 편집 모드 토글
  • 호버로 회색 음표 미리보기
  • 클릭으로 생성 또는 박자 증가
  • Ctrl+클릭으로 박자 감소
  • 더블클릭으로 음표 삭제
  • 음자리표 오른쪽 조표 영역: 호버 후 클릭으로 조표 추가
  • 조표 영역에서 Ctrl 호버: 샵 미리보기
  • 조표 영역 더블클릭: 마지막 조표 삭제
  • 기존 음표 위에서 높낮이를 바꿔 호버 후 클릭하면 음정 변경
  • 기존 음표는 드래그해서 음높이 변경
  • 가사 줄은 직접 클릭해서 수정
  • 좌측 곡 검색에서 제목 또는 가사로 hymn/ccm을 최대 10개까지 찾기
  • 가사 줄에서 Enter: 현재 줄을 둘로 나누기
  • 가사 줄에서 Ctrl+Enter: 현재 위치 이후를 새 슬라이드로 보내기
  • 줄 시작에서 Backspace, 줄 끝에서 Delete: 인접 줄 합치기
  • 절 슬라이드 가사에 <후렴>을 넣으면 후렴 슬라이드로 이동
  • 기존 음표 클릭: 인라인 메뉴 열기
  • 인라인 메뉴의 점 추가/점 제거로 점음표를 편집
  • Ctrl 또는 Shift와 함께 클릭: 같은 줄 선택에 음표 추가
  • 빈 오선지 드래그: 같은 줄의 여러 음표를 박스로 선택
  • 선택 후에는 음표 근처 메뉴에서 바로 적용/해제/취소
  • 연결선 선택은 한 줄씩만 유지되며, 다른 줄을 고르면 이전 선택은 정리
  • 연결선 선택 후 인라인 메뉴에서 바로 묶기 또는 해제
  • 툴바의 슬라이드 추가/삭제로 현재 절 또는 후렴 슬라이드를 조정
  • 내용이 있는 슬라이드를 삭제하면 이전 슬라이드 뒤에 내용이 이어 붙는다
  • Ctrl+Z: 실행 취소, Ctrl+Y: 다시 실행
  • python server.py로 열면 저장/불러오기가 SQLite DB를 우선 사용합니다
